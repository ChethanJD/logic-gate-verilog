\## Gates Implemented

| Gate  | Expression      |

|-------|-----------------|

| AND   | Y = A \& B       |

| OR    | Y = A | B       |

| NOT   | Y = \~A          |

| NAND  | Y = \~(A \& B)    |

| NOR   | Y = \~(A | B)    |

| XOR   | Y = A ^ B       |

| XNOR  | Y = \~(A ^ B)    |
